# System Monitor Script

A simple Bash script designed to monitor core system metrics on Linux/WSL.

## Features
- Displays current RAM usage (Used, Free, Shared).
- Shows CPU load averages over 1, 5, and 15 minutes.
- Checks total and available Disk Space.

## How to Run
1. Make the script executable:
```bash
   chmod +x sys_monitor.sh
./sys_monitor.sh
